# Tripleten web_project_around_react

Projeto front-end do **Around the U.S.**, desenvolvido com **React** e **Vite**, com layout baseado em CSS modularizado (pasta `blocks/`) e componentes React para **perfil**, **cards** e **popups** (editar perfil, editar avatar, adicionar card e visualizar imagem).

## Funcionalidades

- Exibe informações do perfil (nome, descrição e avatar)
- Lista de cards com imagem e título
- Popups:
  - Editar perfil
  - Editar avatar
  - Adicionar novo card
  - Visualizar imagem do card (ImagePopup)

## Tecnologias

- React
- Vite
- JavaScript (JSX)
- CSS (BEM / `blocks`)
- Normalize.css + fontes (Inter)
